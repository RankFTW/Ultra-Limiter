ReLimiter — Feature Reference
==================================

Core Features
-------------

FPS Limit
  Slider 0-500. Quick buttons: 30, 60, 120, 240, Reflex Cap.
  Reflex Cap formula: fps = refresh - (refresh^2 / 3600), floored.
  Example: 165 Hz -> 165 - (27225/3600) = 165 - 7.56 = 157.

Background FPS Limit
  Slider 0-120. Caps frame rate when the game window is not focused.
  0 = auto fallback (fps_limit/3, minimum 10 FPS).
  Uses simple sleep-based limiting, bypasses Reflex pacing.
  All adaptive evaluation frozen while backgrounded.
  Full adaptive state reset on refocus.

VSync Override
  Game (passthrough), On (SyncInterval=1), Off (SyncInterval=0 + ALLOW_TEARING).
  Hooks IDXGISwapChain::Present. Tearing auto-detected via IDXGIFactory5.
  Also applied in Streamline proxy Present hook.

5XXX Exclusive Pacing Optimization
  Hooks IDXGISwapChain2::SetMaximumFrameLatency, overrides to 1.
  Only on waitable swapchains. Game's original value captured and restored.
  Designed for NVIDIA 50-series GPUs with flip metering support.

FG Detection
  Always auto-detected from loaded DLLs. No manual override.
  DLSS FG: nvngx_dlssg.dll, _nvngx_dlssg.dll, sl.dlss_g.dll, dlss-g.dll
  FSR FG: amd_fidelityfx_framegeneration.dll, ffx_framegeneration.dll
  Smooth Motion: NvPresent64.dll
  Multiplier inferred from output FPS / native FPS ratio (2x, 3x, 4x).


Adaptive Features (All Automatic)
----------------------------------

1. Predictive Sleep
   Per-stage StagePredictor instances (GPU, sim, FG) with 8-sample history.
   Weighted averaging (4x/3x/2x/1x). Upward trend detection (3+ frames).
   PipelinePredictor combines stages into total prediction.
   Adaptive safety margin: 500us start, +50us/miss, -10us/30 stable, 200-2000us.
   1:1 mode: tightens interval. FG mode: cadence-based stabilization only.

2. Phase-Locked Timing Grid
   target[k] = epoch + k * interval. No drift accumulation. Late frames snap
   forward. Uses CREATE_WAITABLE_TIMER_HIGH_RESOLUTION + busy-wait tail.

3. Present-to-Present Feedback Loop
   Closed-loop correction. Accumulates P2P error over 30 frames. Nudges
   interval +/-1us per window. Dead zone +/-1us. Clamped +/-8us total.
   Only active without FG. Resets on FPS limit change.

4. Automatic Enforcement Site
   FG active: PRESENT_FINISH (SIM_START only if GPU load < 60%).
   No FG + GPU bottleneck: SIM_START. CPU bottleneck: PRESENT_FINISH.
   65-85% hysteresis band prevents flip-flopping.
   Deferred to PRESENT_BEGIN when FG + queue depth > 1.

5. Queue Pressure Detection
   Monitors OS render queue time from GetLatency. If avg queue time > 1.5x
   target interval, adds 4us to sleep interval. Biases queue depth voting.

6. Adaptive FG Pacing
   Measures actual FG overhead (GPU render end to present end) via GetLatency.
   EMA smoothed (alpha=0.1). Clamped 4-60us. Feeds unified FG adjustment.

7. Cadence Tracking & FG Stabilization
   32-sample CadenceTracker. Rolling mean/stddev of present-to-present deltas.
   Smooth: stddev < 3% target. Jittery: stddev > 8% target.
   2x FG: moderate response, +/-10us range. 3x+ MFG: conservative, never
   tightens below zero. Streak-based state tracking.

8. Unified FG Adjustment
   Combines FG overhead base + cadence correction + GPU headroom + queue stress.
   Single fg_unified_adjust_us value applied to interval. Queue stress prevents
   negative adjustments. MFG-aware conservatism.

9. Dynamic Queue Depth
   Voting-window system (sized to ~2s of frames). Depth 1-3.
   Suggested from safety margin, miss rate, FG state, queue pressure.
   60% supermajority to go deeper, 80% to go shallower.
   4-second hysteresis. FG minimum depth 2.

10. Dynamic Boost Controller
    Tracks GPU idle gaps and steady-state GPU time via EMAs.
    Boost On: consistent idle gaps + post-idle ramp detected.
    Boost Off: GPU load > 85% or thermal throttling suspected.
    Thermal detection: GPU trend rising 3+ sec while sim flat/falling.
    6-second hysteresis. Falls back to game's boost setting when deferring.

11. Bottleneck Detection
    Per-stage EMA tracking: sim, render submit, driver, queue, GPU active.
    40% threshold: if any stage > 40% of total pipeline, flagged as bottleneck.
    Categories: GPU, CpuSim, CpuSubmit, Driver, Queue, None.
    Informs enforcement site selection.

12. Hybrid Queue Wait
    Two-phase: waitable timer sleeps (~0.5ms) then busy-wait for final <1ms.
    50ms timeout. Saves CPU vs pure spin-loop.

13. Time-Based Warmup
    2-second wall-clock warmup from first present. Consistent across frame rates.

14. Proper Reflex Restore
    Captures game's SetSleepMode params on hook. Restores on shutdown.

15. VRR / GSync Awareness
    Ceiling = 3600 * hz / (hz + 3600). 0.5% safety margin. Clamps interval.
    Cached 2s. Disabled when VSync off.

16. Adaptive Interval Adjustment
    GPU load < 70%: -3us. GPU load > 90%: up to +4us.
    Driver shave: always -2us.

17. Flip Metering Block
    Hooks nvapi_QueryInterface. Blocks ordinal 0xF3148C42 (SetFlipConfig).
    Exception: Smooth Motion (needs flip metering).
    Exception: REFramework + Streamline (crash prevention, QI hook skipped).

18. Settings Change Reset
    FPS limit, VSync, or exclusive pacing change triggers full reset:
    pipeline predictor, pipeline stats, boost controller, P2P feedback,
    timing grid, queue depth voting, sleep mode cache. Re-enters warmup.


Interval Computation Pipeline
-----------------------------

Raw interval (from fps_limit / FG divisor)
  -> VRR floor clamp
  -> FG path: unified FG adjustment (overhead + cadence + headroom)
  -> 1:1 path: queue pressure (+4us) + adaptive interval + P2P correction
  -> Driver shave (-2us)
  -> Predictive sleep (1:1 only, can only tighten)
  = Final minimumIntervalUs sent to driver


Reflex Hook Summary
-------------------

SetSleepMode  — Swallowed. Game params captured. UL drives sleep mode.
                Dynamic Boost Controller overrides boost flag when decided.
Sleep         — Swallowed. UL calls Sleep on its own schedule.
SetMarker     — Intercepted. RTSS filtered. INPUT_SAMPLE reordered.
                Timestamps in 64-slot ring buffer. Dedup per frame.
GetLatency    — Not hooked. Called directly for adaptive features.
                Disabled when REFramework + Streamline detected.
QueryInterface — Hooked. Blocks flip metering (SetFlipConfig).
                 Skipped when REFramework detected (crash prevention).


Smooth Motion
-------------

Detected via NvPresent64.dll. NOT the same as DLSS Frame Generation.
When active: Reflex Sleep bypassed, timing-only pacing, flip metering
block disabled, OSD FPS/Native swapped.


OSD Elements
------------

FPS           White   Output frame rate (incl. generated frames)
Native FPS    White   Real render rate from SIM_START markers
Frame         White   Rolling avg frame time (64-frame window)
GPU           Cyan    GPU active render time from GetLatency
Render Lat    Cyan    Sim-start to GPU-render-end
Present Lat   Cyan    Present start-to-end
FG            Gold    Detected FG tech + multiplier
Res           Gold    Render -> output resolution with scale %
Graph         Green   128-sample frametime history, auto-scaled
