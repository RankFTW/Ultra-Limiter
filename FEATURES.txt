Ultra Limiter — Feature Reference
==================================

Core Features
-------------

FPS Limit
  Slider 0-500. Quick buttons: 30, 60, 120, 240, Reflex Cap.
  Reflex Cap formula: fps = refresh - (refresh^2 / 4096), floored.
  Example: 165 Hz -> 165 - (27225/4096) = 165 - 6.64 = 158.

VSync Override
  Game (passthrough), On (SyncInterval=1), Off (SyncInterval=0 + ALLOW_TEARING).
  Hooks IDXGISwapChain::Present. Tearing auto-detected via IDXGIFactory5.
  Also applied in Streamline proxy Present hook.

5XXX Exclusive Pacing Optimization
  Hooks IDXGISwapChain2::SetMaximumFrameLatency, overrides to 1.
  Only on waitable swapchains. Game's original value captured and restored.
  Designed for NVIDIA 50-series GPUs with flip metering support.

FG Mode
  Auto-detects DLSS FG, FSR FG, Smooth Motion from loaded DLLs.
  Manual override: Off, 2x, 3x, 4x.
  Divides target FPS by multiplier for real render rate.

Boost Override
  Game (passthrough), On (force boost), Off (force no boost).
  Controls Reflex Low Latency Boost (GPU clock management).

Pacing Presets
  Native Pacing    — SIM_START, no queue limit. Lowest latency.
  Marker Low Lat   — Reflex markers, max 1 queued frame.
  Balanced         — Reflex markers, max 2 queued frames.
  Stability        — Reflex markers, max 3 queued frames.
  SL Proxy         — Pace generated frames via Streamline proxy.
  Custom           — Manual control of all sub-settings.


UL-Exclusive Adaptive Features
------------------------------

These features are unique to Ultra Limiter. Neither Special K nor Display
Commander implement any of them.

1. Predictive Sleep
   GPU-aware wake scheduling. Tracks 8-sample GPU time history with weighted
   averaging (4x/3x/2x/1x). Detects upward trends (3+ frames rising) and
   biases prediction upward. Adaptive safety margin: 500us start, +50us per
   miss, -10us per 30 stable frames, clamped 200-2000us. Only active when
   GPU-bound. Predictive interval can only tighten the base interval.

2. Phase-Locked Timing Grid
   target[k] = epoch + k * interval. No drift accumulation. Late frames snap
   forward. Uses CREATE_WAITABLE_TIMER_HIGH_RESOLUTION + busy-wait tail.
   ZwSetTimerResolution for max kernel timer precision.

3. Present-to-Present Feedback Loop
   Closed-loop correction. Accumulates P2P error over 30 frames. Nudges
   interval +/-1us per window. Dead zone +/-1us. Clamped +/-8us total.
   Only active without FG. Resets on FPS limit change.

4. Automatic Enforcement Site
   GPU load > 85% -> SIM_START (lowest latency).
   GPU load < 65% -> PRESENT_FINISH (best pacing).
   65-85% hysteresis band prevents flip-flopping.

5. Queue Pressure Detection
   Monitors OS render queue time from GetLatency. If avg queue time > 1.5x
   target interval, adds 4us to sleep interval to throttle proactively.

6. Adaptive DLSS FG Pacing
   Measures actual FG overhead (GPU render end to present end) via GetLatency.
   EMA smoothed (alpha=0.1). Replaces static +24us offset. Clamped 4-60us.

7. Hybrid Queue Wait
   Two-phase: waitable timer sleeps (~0.5ms) then busy-wait for final <1ms.
   50ms timeout. Saves CPU vs pure spin-loop used by SK and DC.

8. Time-Based Warmup
   2-second wall-clock warmup from first present. Consistent across frame
   rates (frame-count warmup is 10s at 30fps but 1.25s at 240fps).

9. Proper Reflex Restore
   Captures game's SetSleepMode params on hook. Restores on shutdown.
   If game never called SetSleepMode, disables Reflex cleanly.

10. VRR / GSync Awareness
    Ceiling = 3600 * hz / (hz + 3600). 0.5% safety margin. Clamps interval
    so limiter never exceeds VRR window. Cached 2s. Disabled when VSync off.

11. Adaptive Interval Adjustment
    GPU load < 70%: -3us (tighten). GPU load > 90%: up to +4us (buffer).
    Driver shave: always -2us to compensate for systematic overshoot.

12. Flip Metering Block
    Hooks nvapi_QueryInterface. Blocks ordinal 0xF3148C42 (SetFlipConfig)
    by returning nullptr. Exception: Smooth Motion (needs flip metering).
    UL's own lookups use original QI pointer. Always active, no toggle.


Interval Computation Pipeline
-----------------------------

Raw interval (from fps_limit / FG divisor)
  -> VRR floor clamp
  -> FG offset (adaptive or +24us static)
  -> Queue pressure (+4us if flagged)
  -> GPU headroom adjustment (-3us to +4us)
  -> P2P feedback correction (-8us to +8us)
  -> Driver shave (-2us)
  -> Predictive sleep (can only tighten)
  = Final minimumIntervalUs sent to driver


Reflex Hook Summary
-------------------

SetSleepMode  — Swallowed. Game params captured. UL drives sleep mode.
Sleep         — Swallowed. UL calls Sleep on its own schedule.
SetMarker     — Intercepted. RTSS filtered. INPUT_SAMPLE reordered.
                Timestamps in 64-slot ring buffer. Dedup per frame.
GetLatency    — Not hooked. Called directly for adaptive features.
QueryInterface — Hooked. Blocks flip metering (SetFlipConfig).


Smooth Motion
-------------

Detected via NvPresent64.dll. NOT the same as DLSS Frame Generation.
When active: Reflex Sleep bypassed, timing-only pacing, flip metering
block disabled, OSD FPS/Native swapped.


OSD Elements
------------

FPS           White   Output frame rate (incl. generated frames)
Native FPS    White   Real render rate from SIM_START markers
Frame         White   Rolling avg frame time (64-frame window)
GPU           Cyan    GPU active render time from GetLatency
Render Lat    Cyan    Sim-start to GPU-render-end
Present Lat   Cyan    Present start-to-end
FG            Gold    Detected FG tech + multiplier
Res           Gold    Render -> output resolution with scale %
Graph         Green   128-sample frametime history, auto-scaled
